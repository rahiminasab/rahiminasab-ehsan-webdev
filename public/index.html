WebAppMaker

This Website allow users to create and use online, mobile friendly, Websites. There are two types of users. We will refer to those users creating the Websites as developers. Users that visit the Websites created by the developers will be referred to as end users. Only developers can create and modify the Websites. End users can not modify the Websites, they can only interact with them. The Website will be built using the MEAN stack using the four underpinning technologies MongoDB, Express, AngularJS, and Node.js.
